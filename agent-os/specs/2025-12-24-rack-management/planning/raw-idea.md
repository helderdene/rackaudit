# Raw Idea

**Feature:** Rack Management — CRUD for racks with location, U-height, power capacity, and visual rack elevation diagram component

## Description
This feature involves creating a complete CRUD (Create, Read, Update, Delete) system for managing server racks. The system should track:
- Rack location information
- U-height (rack unit height)
- Power capacity
- Visual rack elevation diagram component for displaying the rack layout
