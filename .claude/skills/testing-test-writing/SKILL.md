## When to use this skill

- When creating new test files with php artisan make:test --pest
- When editing existing test files in tests/Feature/ or tests/Unit/
- When writing test assertions using Pest's expect() syntax
- When using model factories to create test data
- When mocking external services or dependencies
- When implementing feature tests for HTTP endpoints
- When running tests with php artisan test
- When deciding which user flows to test
- When using Pest datasets for data-driven tests
- When writing browser tests with Pest v4
