## When to use this skill

- When creating or editing API route files (routes/api.php, routes/*.php)
- When building API controllers or resource controllers
- When implementing Eloquent API Resources for JSON responses
- When adding pagination, filtering, or sorting to API endpoints
- When designing RESTful URL structures and choosing HTTP methods
- When setting appropriate HTTP status codes for responses
- When implementing API versioning strategies
- When adding rate limiting to endpoints
- When working with nested resources or related data in APIs
