## When to use this skill

- When adding Tailwind CSS utility classes to HTML or Vue templates
- When editing the main CSS file at resources/css/app.css
- When customizing Tailwind theme in CSS with @theme directive
- When applying consistent spacing, colors, or typography
- When creating responsive layouts with Tailwind breakpoints
- When implementing dark mode styles with dark: variants
- When extracting repeated patterns into reusable components
- When optimizing CSS for production builds
- When working with CSS variables for design tokens
- When styling interactive states (hover, focus, active)
