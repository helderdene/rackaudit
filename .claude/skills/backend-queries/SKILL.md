## When to use this skill

- When writing Eloquent queries in controllers, services, or repositories
- When implementing eager loading with with() to prevent N+1 queries
- When using the query builder for complex database operations
- When optimizing slow queries or analyzing query performance
- When implementing database transactions for atomic operations
- When writing query scopes in models
- When using whereHas, whereIn, or other advanced query methods
- When selecting specific columns instead of SELECT *
- When implementing caching for expensive queries
- When ensuring queries are protected against SQL injection
