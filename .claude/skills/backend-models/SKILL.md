## When to use this skill

- When creating new Eloquent models with php artisan make:model
- When editing existing model files in app/Models/
- When defining model relationships (hasMany, belongsTo, belongsToMany, etc.)
- When implementing accessors, mutators, or attribute casting
- When creating query scopes for reusable query logic
- When setting up $fillable, $guarded, or $hidden properties
- When configuring timestamps, primary keys, or table names
- When implementing model events or observers
- When creating model factories in database/factories/
- When working with soft deletes or other model traits
