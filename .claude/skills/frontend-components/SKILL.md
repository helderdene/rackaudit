## When to use this skill

- When creating new Vue components in resources/js/Components/
- When editing existing Vue single-file components (.vue files)
- When designing component props and their default values
- When implementing component events with emits
- When using slots for flexible component composition
- When extracting reusable logic into composables
- When deciding component boundaries and responsibilities
- When managing component state with ref() and reactive()
- When implementing provide/inject for component communication
- When documenting component usage and prop interfaces
