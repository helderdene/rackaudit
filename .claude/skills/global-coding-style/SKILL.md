## When to use this skill

- When writing new code in any language (PHP, JavaScript, Vue, TypeScript)
- When naming variables, functions, methods, or classes
- When naming files and organizing directory structures
- When deciding how to structure functions or methods
- When applying formatting and indentation standards
- When refactoring code for better readability
- When removing dead code, unused imports, or commented-out blocks
- When extracting reusable functions or avoiding code duplication
- When reviewing code for style consistency
