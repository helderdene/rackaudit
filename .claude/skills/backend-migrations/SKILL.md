## When to use this skill

- When creating new migration files with php artisan make:migration
- When editing existing migration files in database/migrations/
- When adding, modifying, or dropping database tables or columns
- When creating or modifying indexes and foreign key constraints
- When implementing up() and down() methods for reversible migrations
- When planning schema changes for zero-downtime deployments
- When separating schema changes from data migrations
- When naming migration files descriptively
- When considering the order of migration execution
