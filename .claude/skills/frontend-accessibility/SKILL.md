## When to use this skill

- When creating or editing Vue components in resources/js/
- When working on Blade templates in resources/views/
- When implementing forms with proper labels and error messages
- When adding interactive elements like buttons, links, or modals
- When implementing keyboard navigation and focus management
- When choosing between semantic HTML elements (nav, main, article, button)
- When adding ARIA attributes for complex interactive components
- When ensuring sufficient color contrast ratios
- When adding alt text to images or labels to form inputs
- When building modals, dropdowns, or other dynamic UI elements
