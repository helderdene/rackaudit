## When to use this skill

- When implementing mobile-first responsive layouts
- When using Tailwind responsive breakpoint prefixes (sm:, md:, lg:, xl:, 2xl:)
- When creating fluid layouts with percentage-based widths
- When using relative units (rem, em) for scalable sizing
- When designing touch-friendly interfaces with adequate tap targets
- When optimizing images and assets for mobile devices
- When adjusting typography for readability across screen sizes
- When implementing responsive navigation menus
- When testing layouts across mobile, tablet, and desktop viewports
- When prioritizing content display for smaller screens
